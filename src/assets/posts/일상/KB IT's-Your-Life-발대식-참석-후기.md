## 부트캠프 과정 소개

---

### 부트캠프 수업 진행 방식

![과정소개](/assets/img/post_image/2026-03-05/과정소개.png)

- 전공반: 심화 이론과 실습 중심
- 비전공반: 기초 개념과 기본 실습 중심

우선 수업은 전공반과 비전공반으로 나눠서 진행된다고 하셨습니다.

`오전`에는 대표 강사님께서 진행하는 통합 이론 강의가 **Zoom 실시간 수업**으로 진행된다고 하셨습니다.  
`오후`에는 주강사님께서 **대면**으로 반별로 실습 강의를 진행하며 배운 내용을 직접 실습하면서 익히는 방식으로 진행되는 것 같아요.

반은 나누어져 있지만 국민은행에서 요구하는 기술 스택 자체는 동일하기 때문에 전공 여부와 관계없이 **같은 기술을 배우는 구조**라고 합니다.

오전에는 비대면으로 진행되지만 대면반으로 신청한 사람들은 9시까지 강의실로 출석해야 한다고 합니다.  
혹시나 기대했지만... 다시 대면반으로 신청했던 각오를 다졌습니다.

## 커리큘럼

![커리큘럼](/assets/img/post_image/2026-03-05/커리큘럼.png)

전체 커리큘럼은 크게 **4단계 과정**으로 구성되어 있었습니다.

### Step 1 – 프론트엔드 기초

처음 단계에서는 웹 개발의 기본이 되는 **프론트엔드 기술**을 배웁니다.

- HTML / CSS / JavaScript
- Node.js
- Vue.js

웹 인터페이스를 직접 구현해 보면서 **웹 서비스의 기본 구조**를 이해하는 단계입니다.

---

### Step 2 – 백엔드 기초

두 번째 단계에서는 **서버 개발을 위한 기초 기술**을 학습합니다.

- Java
- RDBMS (MySQL)
- JDBC
- MongoDB
- 알고리즘 기초

데이터베이스 설계와 서버 로직 구현 등 **백엔드 개발의 기본 개념**을 배우게 됩니다.

---

### Step 3 – 백엔드 심화

서버사이드 **렌더링** 같이 실제 서비스 개발에 필요한 **백엔드 기술을 더 깊게** 다룹니다.

- Servlet
- JSP
- Spring Framework
- REST API
- 알고리즘 심화

특히 **Spring 기반 웹 서비스 개발**을 배우는 것이 핵심인 것 같아요.

---

### Step 4 – 최종 프로젝트

마지막 단계에서는 지금까지 배운 기술을 활용해 **최종 프로젝트**를 진행합니다.

- AI를 활용한 개발 생산성 향상
- 금융 관련 주제 프로젝트
- 현직자 프로젝트 멘토링

국민은행 부트캠프이다 보니 금융 관련 주제 프로젝트를 진행하게 되는 것 같아요.  
실제 현업 개발자분들에게 피드백을 받으면서 진행할 수 있어서 완성만 잘한다면 좋은 포트폴리오를 만들 수 있을 것 같습니다.

## 평가 기준

![평가기준](/assets/img/post_image/2026-03-05/평가기준.png)

사실 부트캠프를 신청한 이유 중 하나가 우수생이 되면 국민은행 지원 시 **서류 전형 면제** 혜택이 있기 때문이었어요.

우수생이 되기 위한 평가 기준은 다음과 같습니다.

- **출석 30%**
- **평가 40%**
- **태도 15%**
- **수상 15%**

### 출석 (30%)

출석률이 생각보다 크게 반영되는 것 같아요.  
저는 성남에 살아서 세종대까지 1시간 정도 걸리는데 출근 시간이라서 9시까지 도착하려면 넉넉하게 **7시 40~50분**에는 집에서 나와야 도착할 수 있겠더라고요.

지각 3번당 결석 1번으로 처리된다고 하던데 진짜 성실하게 다녀야겠다고 느꼈어요.

---

### 평가 (40%)

가장 비중이 큰 평가 항목입니다.

- 교과목 평가
- 모듈 평가
- 알고리즘 평가

평가는 총 10회가 진행된다고 나와 있는데 **각 평가에서 70점 이상**을 받아야만 우수생이 될 수 있다고 하더라고요.  
우수생이 되기 위해서 공부도 매일 해야겠어요.

---

### 태도 (15%)

- 실습 자료 제출
- 단위 기간 평가
- 수업 참여도

수업을 진행하면서 과제도 주어진다고 하셨어요.  
과제는 물론 수업 때도 더 적극적으로 임하고 질문하려고 합니다.

---

### 수상 (15%)

프로젝트 결과에 따라 **수상 점수**도 평가에 반영됩니다.  
프로젝트 결과가 좋다면 최우수상도 받고 우수 수료생으로도 이어질 수 있을 것 같습니다.

프로젝트도 열심히 해야겠네요 🔥🔥
